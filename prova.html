Lista de exercícios - POO

  1. Construtores são utilizados para:

    • a. Construir interfaces gráficas
    • b. Liberar memória
    • c. Inicializar um objeto recém-criado
    • d. Criar uma sub-classe

  2. Qual o termo utilizado para acessar o método ou as variavéis de uma superclasse?

    • a. Super
    • b. Using
    • c. Is_a
    • d. Has_a

  3. Quando a subclasse declara um método que possui o mesmo tipo de argumentos que um método declarado por uma das suas superclasses, é chamado de:

    • a. Método overriding
    • b. Método overloading
    • c. Operador overloading
    • d. Operador overriding

  4. Um objeto é uma instância de um(a):

    • a. Parâmetro
    • b. Classe
    • c. Método
    • d. Aplicação

  5. O processo no qual objetos de uma classe recebem as propriedades de objetos de uma outra classe é chamado de:

    • a. Polimorfismo
    • b. Herança
    • c. Encapsulamento
    • d. Associação
