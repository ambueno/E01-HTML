Variáveis e expressões matemáticas

Conceitos fundamentais:

  • Uma variável é caracterizada por um local reservado na memória para armazenar um tipo de dado;
  • Toda variável deve ter um identificador, ou seja um nome;
  • Toda variável deve ter um tipo;
  • Em C, todas as variáveis devem ser declaradas antes de serem usadas.

Tipos:

Tipo de dado é um sistema utilizado para definir diversas propriedades básicas sobre o dado armazenado na memória. propriedades essas como tipo do dado, seu intervalo, bytes ocupados, como esses bytes serão interpretados, etc.

Em C, os tipos de dados podem ser agrupados em 3 grandes categorias:

  1. Tipos primitivos

    2. A linguagem C suporta quatro tipos primitivos int, char, ponto-flutuante (float e double) e void.
    Tipos primitivos também são chamados de pré-definidos ou tipos de dados básicos.

    O tamanho e a faixa de valores de um tipo de dado depende da máquina e pode variar de compilador para compilador. O C padrão requer apenas o minimo tamanho a ser preenchido por cada compilador para cada tipo de dado.
    Por exemplo, o tamanho de int varia em diferentes compiladores, mas precisa ser no mínimo 2 bytes em todos os compiladores.

    • int: O termo int é utilizado para definir um tipo numérico inteiro. O tamanho de um int é de 2 ou 4 bytes (depende do compilador) e a faixa de valores que pode armazenar varia de -32768 a 32767 ou -2147483648 a +2147483647.
    • float: Um valor real é definido pelo termo float ou double. Float caracteriza um número real de precisão simples. Seu tamanho é de 4 bytes e sua faixa de valores vai de 1.2E-38 a 3.4E+38, apresentando, portanto, uma precisão de até 6 casas decimais
    • double: O termo double caracteriza um número real de precisão dupla. O tamanho de um double é de 8 bytes e sua faixa de valores vai de 2.3E-308 a 1.7E+308, apresentando, portanto, uma precisão de até 15 casas decimais.
    • char: Todo valor de um único caracter em C é representado usando char. Contudo, na realidade são armazenados valores inteiros ao invés dos caracteres em si, de modo que o computador tem que mapear cada inteiro com o caracter correspondente usando um código numérico.
      O mais comum dentre os códigos numéricos é a convenção ASCII.
      O tamanho de um char é um byte e a faixa de valores que pode armazenar vai de -128 a 127.
    • void: Enquanto tipo de dado, o termo void representa a ausência de valores ou operadores, sendo utilizado para representar nada, vazio. Ou seja, internamente nada é armazenado.
      Além disso, também é frequentemente utilizado para representar ponteiros genéricos ou o tipo de retorno de uma função, nesse caso de uma função onde o valor de retorno é vazio.

    Qualificadores de tipos de dados

    Qualificadores são opcionalmente adcionados nos tipos primitivos de dados conforme necessário. Em síntese, alteram o comportamento e propriedades dos tipos de dados.
    Existem três tipos de qualificadores de dados em C, tamanho, sinal e tipo. A sintaxe padrão de uso de um qualificador é:

                                [qualificador-de-sinal] [qualificador-de-tamanho] <tipo-de-dado-primitivo>

    • Qualificadores de tamanho

      • Qualificadores de tamanho são utilizados para alterar o tamanho do tipo de dado primitivo. C dá suporte aos tipos short, long e long long.
      Geralmente, qualificadores de tamanho são utilizados com o tipo int. Contudo, o tipo double também suporta o qualificador long.
      São eles:

      • short: Um short ocupa pelo menos 2 bytes e deve ser menor ou igual a um int.
      • long: Um long ocupa pelo menos 4 bytes e deve ser maior ou igual a um int.
        Ou quando sendo aplicado a um double sua precisão deve ser maior ou igual a de um double sem qualificador, definindo um ponto-flutuante de dupla precisão com um mínimo de 12 bytes.
      • long long: Long long é suportado pelo compilador de C GCC, ocupa pelo menos 8 bytes e deve ser maior ou igual a um long.

    • Qualificadores de sinal

      • Qualificadores de sinal são utilizados para especificar a natureza de tipo, isto é, se a variável pode ter valores negativos ou não.
      Podem ser utilizados com inteiros (int) ou com caracteres (char). C suporta dois qualificadores de sinal, signed (com sinal) and unsigned (sem sinal).
      Signed especifica uma variável que pode conter tanto valores positivos, quanto negativos. Já unsigned especifica uma variável que só pode contar valores positivos.

      • signed: valor com sinal, isto é, a variável pode conter valores positivos ou negativos.
      • unsigned: valor sem sinal, isto é, a variável pode conter apenas valores negativos.
